# 📖 小说转换器 (Novel Converter)

## ✨ 简介
[English ](README_en.md)

欢迎使用小说转换器！这个Python脚本可以轻松地将TXT格式的小说文件转换为EPUB格式，支持章节划分和封面添加。无论你是想将自己的小说发布为电子书，还是想将喜欢的小说转换为便于阅读的格式，这个工具都能满足你的需求。

## 📦 依赖
在开始之前，请确保安装了`EbookLib`库。你可以通过以下命令进行安装：
```bash
pip install EbookLib
```

## 🛠️ 使用方法
1. **准备文件**：
   - 将你的小说文本文件命名为`novel.txt`，并将封面图片命名为`cover.jpg`，放在与脚本相同的目录下。
   - 确保TXT文件的章节之间用两个换行符分隔，以便程序能够正确识别章节。

2. **配置环境变量**：
   - 创建一个名为`.env`的文件，内容如下：
     ```env
     TXT_FILE=./novel.txt
     COVER_IMAGE=./cover.jpg
     TITLE=你的小说标题
     AUTHOR=作者名
     ```

3. **运行脚本**：
   - 在终端中运行以下命令：
   ```bash
   python src/main.py
   ```

4. **输出文件**：
   - 转换完成后，输出的EPUB文件将命名为你在`.env`文件中指定的标题，例如`你的小说标题.epub`。

## ⚠️ 注意事项
- 确保TXT文件的章节之间用两个换行符分隔，以便程序能够正确识别章节。
- 封面图片应为JPEG格式，命名为`cover.jpg`。

## 📖 示例
在`src/main.py`中，示例调用如下：
```python
create_epub('novel.txt', 'cover.jpg', '小说标题', '作者名')
```
你可以根据需要修改标题和作者名。
